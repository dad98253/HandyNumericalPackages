RKF45 - Runge-Kutta-Fehlberg ODE Solver

RKF45 is the Watt and Shampine implementation of the Runge-Kutta-Fehlberg ordinary differential equation solver, with automatic error estimation.

Files you may copy include:

  • RKF45.F90, the source code
  • RKFPRB.F90, a sample calling program
  • RKFPRB.OUT, the output from the sample calling program

The list of routines includes:

  • RKF45 carries out the Fehlberg fourth-fifth order Runge-Kutta method.
  • RKFS implements the Fehlberg fourth-fifth order Runge-Kutta method.
  • FEHL takes a single Fehlberg fourth-fifth order Runge-Kutta step.
